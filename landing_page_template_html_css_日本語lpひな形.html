公開までの3ステップ

1. リポジトリ作成

GitHubで新規リポジトリを作成し、この index.html をアップロード。

2. 画像差し替え

/images にヒーローやロゴを配置し、altテキストを適切に設定。

3. Pages公開

Settings → Pages で main ブランチを指定すればURLが発行されます。

よくある質問

画像はどこに置けばよいですか？

/images フォルダを作成し、hero.jpg やロゴ画像を配置してください。HTML内の img の src を差し替えます。

フォーム送信はできますか？

HTML+CSSのみのため、フォーム送信には外部サービス（Googleフォーム等）か、JS/バックエンドの追加が必要です。

SEO対策はされていますか？

title/description/OG/Twitterカードを定義済みです。実運用ではページ速度・本文構造・見出し最適化も調整してください。
